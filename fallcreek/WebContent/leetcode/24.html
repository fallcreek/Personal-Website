24. Swap Nodes in Pairs

Given a linked list, swap every two adjacent nodes and return its head. For example,Given 1->2->3->4, you should return the list as 2->1->4->3. Your algorithm should use only constant space. You may not modify the values in the list, only nodes itself can be changed.

Difficulty: Easy

Tag: Linked List

My Solution

Algorithm

The problem requests to use constant space, so I create three instances of ListNode, which is defined bellow.

/**
* Definition for singly-linked list.
* public class ListNode {
*     int val;
*     ListNode next;
*     ListNode(int x) { val = x; }
* }
*/

They are previousNode, currentNode and nextNode.

Then follow the steps bellow:

  1. Input a list.
  2. Separate the list into two parts: the first two nodes and the rest.
  3. Swap the two nodes in the first part.
  4. Link the original first node in the first part to the second node (if exists) in the second part.
  5. Input the second part and return to step 2.

Quit conditions and special circumstances will be revealed in code.

A rough sketch of the algorithm:


Code

public class Solution {
    public ListNode swapPairs(ListNode head) {

    boolean findReturnNode = false;
    ListNode previousNode = null;
    ListNode currentNode = head;
    ListNode nextNode = null;

    while(currentNode != null && currentNode.next != null)
    {
        if(!findReturnNode)
        {           
            head = currentNode.next;
            findReturnNode = true;
        }
        if(previousNode != null)
        {
            previousNode.next = currentNode.next;
        }

        nextNode = currentNode.next.next;
        currentNode.next.next = currentNode;
        currentNode.next = nextNode;
        previousNode = currentNode;
        currentNode = nextNode;  
    }
    return head;
    }
}
