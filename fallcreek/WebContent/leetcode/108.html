108. Convert Sorted Array to Binary Search Tree

Given an array where elements are sorted in ascending order, convert it to a height balanced BST.


Difficulty:Â Medium

Tag: Tree, Depth-first Search

My Solution

Algorithm

If we pick out the middle node of the given array to be the root and the nodes before it in the arrary become its left children and the nodes after it become its right children, the number difference of nodes on both side of the root won't exceed one. Do it recursively, we will get a height balanced BST.

For example,

Code

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
public class Solution {
    public TreeNode sortedArrayToBST(int[] nums) {
        if(nums == null || nums.length == 0)    return null;
        
        return balance(nums,0,nums.length - 1);
        
    }
    
    public TreeNode balance(int[] nums, int start, int end)
    {
        if(start > end) return null;
        
        int mid = start + (end - start) / 2;
        TreeNode head = new TreeNode(nums[mid]);
        
        head.left = balance(nums,start,mid-1);
        head.right = balance(nums,mid+1,end);
        
        return head;
    }
}
