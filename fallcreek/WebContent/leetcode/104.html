104. Maximum Depth of Binary Tree

Given a binary tree, find its maximum depth.

The maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.

Difficulty:Â Easy

Tag: Tree, Depth-first Search

My Solution

Algorithm

The maximum depth of a tree equals one plus the larger maximum depth of the root's two subtrees. If a subtree is null, its depth is zero.

Code

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
public class Solution {
    public int maxDepth(TreeNode root) {
        if(root == null)    return 0;
        
        return 1 + Math.max(maxDepth(root.left),maxDepth(root.right));
    }
}

Detail

Complexity

  • Time Complexity: O(n). Because the function will be called once for each node in the tree.
  • Space Complexity: The memory used for recursion is decided by the given tree. In the worst case, each node in the tree only has one subtree. Then the space complexity is O(n).
