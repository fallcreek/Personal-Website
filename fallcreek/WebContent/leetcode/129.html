129. Sum Root to Leaf Numbers

Given a binary tree containing digits from 0-9 only, each root-to-leaf path could represent a number.

An example is the root-to-leaf path 1->2->3 which represents the number 123.

Find the total sum of all root-to-leaf numbers.


For example,

   1
  / \
 2   3

The root-to-leaf path 1->2 represents the number 12.

The root-to-leaf path 1->3 represents the number 13.

Return the sum = 12 + 13 = 25.

My Solution

Algorithm

A recursive DFS algorithm can solve it easily. We need to pass two variables to the next recursion. They are the total sum of all paths and the sum of current path. When we reach a leaf, the sum of current path will be added to the total sum. Do it recursively unitl we have examined all nodes.

Code

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
public class Solution {
    public int sumNumbers(TreeNode root) {
        return DFS(root,0,0);
    }
    
    public int DFS(TreeNode root, int sum, int total)
    {
        if(root == null)    return total;
        
        sum = sum * 10 + root.val;
        if(root.left == null && root.right == null) return total + sum;
        
        total = DFS(root.left,sum,total);
        total = DFS(root.right,sum,total);
        return total;
    }
}
